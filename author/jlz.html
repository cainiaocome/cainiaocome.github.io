How to detect disk is mounted or not in bash

Published: Fri 20 November 2020
By jlz

In Linux.

I mounted two disks on my raspberrypi via USB3.0 port. But the port, hub, or the wire might broke so the mounting action might fail. If it fails, the backup script might fail so I want to check the disk is mounted successfully before backup my raspberrypi.

Show you the code:

    uuid="836a6e36-4b6c-437a-aa52-e1fba22093d1"
    mountpoint=$(lsblk -o UUID,MOUNTPOINT | awk -v u="$uuid" '$1 == u {print $2}')
    if [[ -n $mountpoint ]]
    then
        echo "backup disk mounted, backup now..."
        rsync -avr /root/ /disk2/rasp_backup/
    else
        echo "backup disk not mounted, exit now..."
    fi
